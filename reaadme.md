Exercícios fáceis



1. Contar quantos números são maiores que a média



Receba 5 números inteiros do usuário e armazene em um array.
Calcule a média desses números.
Conte e imprima quantos números são maiores que a média.





2. Substituir valores negativos por zero



Receba 6 números inteiros do usuário em um array.
Substitua todos os valores negativos do array por zero.
Imprima o array modificado.





Exercícios intermediários



3. Encontrar maior e menor valor



Receba 6 números inteiros do usuário para preencher um array.
Encontre e imprima o maior e o menor número do array



4. Contar números pares e ímpares



Receba 8 números inteiros do usuário e armazene em um array.
Conte e mostre quantos números pares e quantos números ímpares existem no array.





Exercícios difíceis



5. Inverter um array



Receba 10 números inteiros do usuário e armazene em um array.
Inverta a ordem dos elementos do array **in-place** (sem usar array auxiliar).
Imprima o array invertido.





6. Remover elementos duplicados



Receba 10 números inteiros do usuário e armazene em um array.
Remova os valores duplicados do array (mantenha apenas uma ocorrência de cada número).
Imprima o array sem duplicatas.
